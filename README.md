# PyramidKernel
Just a simple kernel.

## Build requirements
In order to build PyramidKernel, make sure to have the following installed:
- git
- bash
- make
- GCC 
- nasm
- QEMU (to test it)

### Build
```
make
```
### Run in QEMU:
```
make run
```
### Compile to ISO
```
make iso
```
### Bochs
```
make bochs
```
